### Introduction

The HL7 v2+ specification provides a computable representation of the HL7 Version 2 standard using FHIR StructureDefinition resources.

### Purpose

This Implementation Guide represents the HL7 V2 standard in a modern, computable format that enables:

- Machine-readable definitions of data types, segments, and messages
- Formal validation of V2 message content
- Integration with FHIR-based tooling and ecosystems
- Preservation of the normative V2 specification text

### Contents

The main sections of this IG are:

* [Introduction](introduction.html) - Background and purpose of V2+
* [Control](control.html) - Message control and structure rules
* [Data Types](data-types.html) - Primitive and complex data type definitions
* [Domain Messaging](domains.html) - Domain-specific messages and events
